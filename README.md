# KD-Tree-3D
OpenGL을 이용한 3차원 KD트리  
파티클의 개수는 20000개  

**사용 API**  
- OpenGL  
- GLUT  


**시뮬레이션 환경**  
- CPU : intel i7-10 10700K  
- GPU : RTX 3070 BLACK EDITION OC D6 8GB  
- RAM : samsung DDR4-3200(8GB) x 2


**컨트롤**  
- 마우스 좌클릭 : 화면 이동 회전
- 마우스 휠클릭 : 화면 상하좌우 이동  
- 마우스 우클릭 : 화면 줌  
- S : 쿼리 시행
